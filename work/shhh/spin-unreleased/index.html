03. SPIN UNRELEASED

Website, 2021

SPIN Unreleased is a hidden section of the new SPIN website currently being developed. Since work often cannot be publicly shared, this proposed new space will function as a private view to allow the studio to give select partners a glimpse into unreleased and yet-to-be-released work. Access is granted through a secret code entered through the homepage of the public website; pressing and holding the letters s, p, i, and n on desktop, or drawing an s on mobile will onlock the password protected page. The password is unique to each invitee, and so upon entry they are greeted with a personalised message, along with a specially curated version of the space.
